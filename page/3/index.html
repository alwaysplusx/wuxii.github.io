JPA基础配置

发表于 2016-12-31 | 分类于 javaee ， jpa |

JPA是JavaEE规范中的一员, 最新规范版本号为JPA 2.1. JPA对JavaBean的持久化进行了规范化的描述, 定义了一整套完善的ORM映射和数据库操作的规范, 使得开发人员不再使用复杂繁琐的JDBC来操作数据. JPA配置主要体现在persistence.xml中, 其配置文件默认位于类路径下META-INF/persistence.xml.
JPA只定义规范, 其功能的实现还是依靠具体的厂商, 现主要使用的JPA提供方有

  • Hibernate
  • Eclipselink
  • OpenJPA

Maven私服搭建

发表于 2016-12-29 | 分类于 maven |

Apache Maven是当今主流的Java自动化构建工具, maven通过连接到中央库来下载所需要的依赖包, 中央库是国外网站访问速度受到限制. 一般公司内部都通过搭建自己的私服来代理对中央库的请求, 公司私服既可以满足公司内部的需求又可以减少对中央库的请求次数.

常见的私服搭建与管理软件有:

  • Sonatype Nexus
  • Apache Archiva
  • JFrog Artifactory

Maven基础介绍

发表于 2016-12-29 | 分类于 maven |

Apache Maven?

Apache Maven是用于Java项目自动化构建的工具, 其主要有两个目的, 其一描述项目是如何构建的; 其二描述项目的整体依赖. 工程主要通过XML文件描述其构建过程以及构建所需要的依赖.

Maven自动从Maven Central Repository下载Java libraries和相关插件包. Maven采用插件架构, 用户可以根据自己的需求独立开发相应的插件.

OAuth2.0

发表于 2016-12-27 | 分类于 security ， oauth2.0 |

OAuth2.0主要用于对第三方的授权, 当第三方需要访问资源服务器时需要向授权服务器申请授权得到用户的授权确定后服务器给予对应资源服务器的授权token, 使得第三方可以在规定的时间和权限内访问用户的资源.
