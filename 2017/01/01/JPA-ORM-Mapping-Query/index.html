JPA ORM Mapping & Query

发表于 2017-01-01 | 分类于 javaee ， jpa |

Object Relation Mapping(ORM), JPA ORM Mapping是为关系型数据库提供表与对象的映射解决方案. JPA ORM Mapping提供有基于XML与注解(推荐)的两种方式.

Jpa关系配置

Jpa关系配置主要通过注解(推荐)或XML文件进行配置, 主要的注解有:

  • javax.persistence.ManyToOne
  • javax.persistence.ManyToMany
  • javax.persistence.OneToMany
  • javax.persistence.OneToOne

相关的配置在官方Javadoc中有详细说明.

JPA事件API

JPA Entity Event是基于注解的一套Entity操作实践API, 通过为实体或为其指明@EntityListeners对应的事件监听即可完成对实体的操作监听.

  • javax.persistence.PrePersist - 保存前触发
  • javax.persistence.PostPersist - 保存后触发
  • javax.persistence.PreRemove - 删除前触发
  • javax.persistence.PostRemove - 删除后触发
  • javax.persistence.PreUpdate - 更新前触发
  • javax.persistence.PostUpdate - 更新后触发
  • javax.persistence.PostLoad - 加载后触发
  • javax.persistence.EntityListeners - 指定外部的监听Listener
